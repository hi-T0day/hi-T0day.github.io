Hexo是什么？

问题分析：Hexo的概述 -> Hexo的部署方式 -> Hexo的优点

Hexo是一款基于Node.js的一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页，你可以把生成的静态网页上传到 Web 服务器上，这里我们选用的是 GitHub 做 Web 服务，然后就可以访问到博客站点，因为每个页面都是静态的，所以访问速度是非常快的，用户体验非常好。

如何安装Hexo？

问题分析: Hexo基础环境怎么安装 -> Hexo 如何在Github上展示 -> Concise模版怎么安装？-> 相关配置文件的设置

Hexo基础环境安装（以Mac为例）

  • 首先下载 nodejs 环境
  • 安装Hexo框架包:
    • npm install -g hexo-cli
  • 创建站点:
    • hexo init my-blog

Github部署Hexo

  • 在github中新建一个repo, repo的名字为 your-blog-name.github.io
  • 安装hexo-deployer-git, 从而可以利用hexo deploy命令将内容上传到刚刚建立的repo中:
    • npm install --save hexo-deployer-git

Concise模版的安装

Concise是本站所用的模版

  • 模版下载到本地:
    • git clone git@github.com:sanonz/hexo-theme-concise.git themes/concise
  • 在 \_config.yml 配置文件中把theme值改为concise.
  • 复制 themes/concise/\_config.example.yml 配置文件为 themes/concise/\_config.yml
  • 为了更好生成静态文件, 安装Hexo编译器hexo-renderer-less:
    • npm install hexo-renderer-less --save

修改配置文件

  • 网站根目录下的Hexo配置文件_config.yml,可以修改网站信息,比如title, author, url等信息

  • concise模版下的配置文件themes/concise/_config.yml, 可以修改网站页面模版展示的内容。比如修改: author信息, gittalk密钥, menu菜单.
